# GPA/CGPA Calculator

A simple GPA and CGPA calculator built with Streamlit for Nigerian university students. The app supports both 4.0 and 5.0 scales, works for freshers and returning students, and includes a planning tool for achieving a target CGPA.

---

## Supported Features

- GPA and CGPA calculations
- Works with 4.0 and 5.0 grading scales
- Multiple semesters and courses
- Freshers can calculate semester GPA
- Returning students can calculate CGPA
- Suggestion tool for reaching a target CGPA with different score patterns
- Clean interface with unique widget keys
- Banner support on pages

---

## Grading System

### 4.0 Scale
| Grade | Points |
|------|--------|
| A | 4.0 |
| B | 3.0 |
| C | 2.0 |
| D | 1.0 |
| F | 0.0 |

### 5.0 Scale
| Grade | Points |
|------|--------|
| A | 5.0 |
| B | 4.0 |
| C | 3.0 |
| D | 2.0 |
| E | 1.0 |
| F | 0.0 |

## Pages

### Homepage
Overview of the application and how to use it.

### GPA/CGPA Calculator
- Calculate GPA per semester
- Calculate CGPA (if user has previous CGPA)
- Works with either 4.0 or 5.0 scale

### What Do I Need To Get?
Planning tool to meet a target CGPA with different realistic score distribution options:
- Higher first semester, lower second semester
- Lower first semester, higher second semester
- Balanced effort for both semesters
